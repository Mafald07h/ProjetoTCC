Projeto de desenvolvimento de uma ferramenta online para a divulgação de jogos, com acessibilidade a informação. O projeto foi orientado pelo professor <a href="https://integra.ifpb.edu.br/portfolio/pessoas/cicero-alves-da-silva/apis">Cícero Alves da Silva</a>, e sendo aprovado com nota 100. Além disso, o mesmo projeto foi apresentado na Semana de Ciência,Tecnologia e Cultura do IFPB-CR na sua 10ª edição, e novamente aprovado, tendo uma nota de 76,5.
